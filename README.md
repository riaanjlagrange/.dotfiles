# Dotfiles

My personal dotfiles managed with GNU Stow.

## Managed Applications

- Flameshot
- GitHub CLI
- Git
- Hyprland
- Kitty
- Neovim
- Spotify
- SSH
- Starship
- Tmux
- Waybar
- Wofi
- Zsh

## Setup

```bash
git clone https://github.com/riaanjlagrange/.dotfiles ~/.dotfiles
~/.dotfiles/bin/setup.sh
```
